Project: Analysis of Clustering and Dimensionality Reduction
Techniques For Classification Problems
This README provides instructions on how to retrieve the code, data, and dependencies, and how to run the code for the project "Analysis of Clustering and Dimensionality Reduction
Techniques For Classification Problems".


1. Retrieving the Code
GitHub Repository:
Clone the repository using the following command:
sh
Copy code
git clone https://github.com/ManaviShukla/ml_assignment3.git
Alternatively, you can download the ZIP file of the repository from the GitHub page and extract it to your desired directory.

Diabetes Health Indicators Dataset: It is being extracted in the code itself using ucimlrepo module
MAGIC Gamma Telescope: It is being extracted in the code itself using ucimlrepo module

Downloading the Data:
Download the datasets from the provided links.
